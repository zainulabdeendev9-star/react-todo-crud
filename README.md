# React Todo CRUD App

## Overview
This project is a simple Todo application built with React to
practice core concepts like state management, component
re-rendering, and immutable updates.

## Features
- Add new todos
- Mark todos as completed
- Delete todos
- Controlled form inputs
- Conditional UI rendering

## Concepts Practiced
- useState
- Component-based architecture
- State immutability
- Event handling
- List rendering with keys

## Tech Stack
- React
- JavaScript (ES6)
- CSS

## Future Improvements
- Persist todos using localStorage
- Add edit functionality
- Convert to full-stack MERN app
